0. 개인 홈페이지/블로그 제작 (http://kimmyungjoong.github.io)

개인 홈페이지 포트폴리오 개요
주소 http://kimmyungjoong.github.io
레이아웃 코딩 방식 Jekyll, Bootstrap
제작기간 1 weeks
제작참여도 Jekyll 테마 커스터마이징
설명

- 포털에서 제공되는 블로그 사용하지 않고 직접 제작
- jekyll 테마를 활용하여 개인 홈페이지 커스터마이징(깃헙을 이용한)
- 다양한 언어를 접하고 새로운 것을 공부하는것은 언제나 흥분된다
- 항상 공부하고 연구하여 크레이티브한 UI를 개발하고 연구하는 습관을 길들이자

1. 네이버 채용 사이트 제작 (http://recruit.navercorp.com/naver/recruitMain)

네이버 채용 포트폴리오 개요
주소 http://recruit.navercorp.com/naver/recruitMain
레이아웃 코딩 방식 HTML5, CSS, JQUERY
제작기간 4 weeks
제작참여도 마크업 100%, Jquery mobile css customazing
설명

- css3를 활용한 애니메이션 효과 적용.
- 개발자가 사용하는 라이브러리에서 사용되는 css를 커스터 마이징 (해당 프로젝트에 해당되지 않는 css를 걸러내기)
- 디자이너가 요구하는 인터랙션 협의 (크로스 브라우징 범위를 정의하고 css3 만으로 인터랙션 구현)
- 방대한 컨텐츠 내용을 공통으로 정의하여 구현 (직군별 이야기 컨텐츠를 공통UI로 제작)
- 협업 부서간의 커뮤니케이션이 최고였던 프로젝트

3. 네이버 스토어 사이트 제작 (http://naverstore.naver.com)

네이버 스토어 포트폴리오 개요
주소 http://naverstore.naver.com
레이아웃 코딩 방식 HTML5, CSS
제작기간 3 weeks
제작참여도 마크업 100%, Flexible UI
설명

- 가로 해상도 1280px일 경우와 1920px 이상일때 body에 class="wide" 삽입/제거 제어
- 이미지 영역이 3단으로 컬럼으로 나누어질때 %적용시 크로스브라우징 이슈 해결(ie7까지 대응)
- 서브 페이지 가변에 따라 가운데 정렬
- 제품 상세 페이지 이미지 중앙 리사이즈
- faq 페이지 클래스 하나로 질문 답변 on/off 제어
- 생각을 한번 더 할수록 class 제어 하는 부분이 줄어들었다.

6. 라인 제휴 제안 사이트 (https://partner.line.me)

라인 제휴 제안 포트폴리오 개요
주소 https://partner.line.me
레이아웃 코딩 방식 HTML5, CSS
제작기간 10 MD
제작참여도 마크업 100%, 다국어
설명

- body에 나라별 lang을 치환하여 다국어 진행(9개국어)

- 텍스트 최대 길이에 대한 고민(버튼 안에 들어가는 가장 긴 텍스트를 기준으로 작업)
- 다국어를 진행할 때 폰트와 컨텐츠 길이에 대한 고민이 우선으로 필요하다
- 각 나라별 기본으로 쓰는 폰트를 조사한것도 좋은 경험이 되었던 프로젝트
